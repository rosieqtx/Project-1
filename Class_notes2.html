Lesson 1

What's a web page?

A web page is a text document. The language used is primarily HTML.

How Coding Works

Coding is talking to the computer in a language the computer understands. It needs to be exactly right because computers are stupid.

Basic HTML Vocabulary

  • Tag: An HTML tag is what the computer takes direction from. Most tags have an opening tag. Some tags are called "void" tags and do not require a closing tag.
  • Element: An HTML element is everything within opening and closing tags.
  • Attribute: This is a property of an HTML element, put inside a tag.

How to break a design into boxes - because everything on the web is a box

Box Sizing

HTML elements are boxes and each box has 4 components. These are the box, padding, border, and margin.

It is hard to size boxes acurately.

  • Set sizes in terms of percents rather than pixels.
  • Set the box-sizing attribute to border-box for every element.

Box Positioning (Notes directly taken from class slide.)

Divs are block elements (as opposed to inline), so by default they take up the entire width of a page.
Adding the rule display: flex; to the appropriate CSS will override this behavior and let divs appear next to each other.

Structure

HTML and CSS follow a logical structure, where each element has attributes and each subelement either has its own attributes or inherits the attributes of the element above it. This can be diagrammed in a tree shape.

Avoiding repetition

One key element of programming is avoiding repetition. In this document, one way I avoided repetition by using CSS to style the H2 tag rather than using the b tag and styling it everytime I wanted a subheading. CSS classes are another great way to avoid repetition.
